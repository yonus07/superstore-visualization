Superstore Data Visualization Project

This project uses Tableau to analyze the Superstore dataset and create simple, clear visualizations that tell a meaningful business story.

What’s Included
- Tableau dashboards
- Key insights from sales, profit, customers, and discount analysis
- Final PDF report and PPTX presentation

Key Insights
- Technology category has the highest sales
- Some Furniture sub-categories show losses
- West region performs the best overall
- Top customers contribute a major share of revenue
- Higher discounts reduce profit

Files
- `/data` → Dataset used
- `/images` → Tableau screenshots
- `/reports` → PDF & PPTX reports

Tools Used
